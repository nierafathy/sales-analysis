# مشروع تحليل المبيعات (Sales Analysis)

## وصف المشروع
هذا المشروع عبارة عن تحليل بيانات مبيعات لمنتجات مختلفة عبر عدة شهور. الهدف هو معرفة المنتجات الأعلى والأقل مبيعًا، وتحليل اتجاه المبيعات لكل شهر.

## الأدوات المستخدمة
- Excel
- Pivot Tables
- Charts
- Conditional Formatting

## خطوات التحليل
1. عمل Pivot Table لتجميع المبيعات لكل منتج لكل شهر.
2. رسم Charts لتوضيح Trend المبيعات لكل منتج.
3. استخدام Conditional Formatting لتسليط الضوء على أعلى وأقل المبيعات.

## Insights
- المنتج A حقق أعلى مبيعات في معظم الشهور.
- المنتج C حقق أقل مبيعات.
- شهر Feb كان الأفضل في المبيعات مقارنة بشهر Jan.
